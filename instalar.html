Instalación de Yacaré para desarrollo

En Linux

  1. Es necesario instalar los siguientes paquetes: php, php-mysqlnd, php-mbstring, php-intl, subversion, mysql-server. En Ubuntu los paquetes php-* pueden llamarse php5-*. El paquete mysql puede remplazarse por el paquete mariadb.
  2. Descargar desde la página de Symfony la versión estándar más reciente (Symfony_Standard_Vendors_2.3.7.tgz al momento de escribir este documento).
  3. Abrir una consola de comandos para los siguientes pasos.
  4. Crear una carpeta para la aplicación e ingresar a la misma, con los siguientes comandos:
  • mkdir yacare
  • cd yacare
  1. Obtener Yacaré desde el repositorio SVN, con el siguiente comando:
  • svn co http://webmuni/svn/yacare/trunk ./
  1. Descomprimir la distribución de Symfony descargada, sin remplazar los archivos existentes:
  • tar --strip-components=1 -zxkf /carpeta_de_descarga/Symfony_Standard_Vendors_2.3.7.tgz Symfony
  1. Descargar Composer:
  • php -r "eval('?>'.file_get_contents('https://getcomposer.org/installer'));"
  1. Instalar y actualizar los bundles:
  • php composer.phar update
  1. Crear la base de datos y el usuario de acceso:
  • mysql -uroot
  • CREATE DATABASE yacare;
  • GRANT ALL ON yacare.* TO 'yacare'@'localhost' IDENTIFIED BY '123456';
  • GRANT ALL ON yacare.* TO 'yacare'@'%' IDENTIFIED BY '123456';
  • EXIT
  1. Si se dispone de datos de prueba, se los puede incorporar:
  • mysql -uyacare -p123456 --database=yacare < datos_de_prueba.sql
  1. Actualizar las estructuras de datos
  • php app/console doctrine:schema:update --force
  1. Ejecutar el servidor web de desarrollo:
  • php app/console server:run

En Windows

  1. Descargar e instalar su distribución favorita de Linux, por ejemplo Fedora.
  2. Continuar desde el paso 1 de la sección anterior.
